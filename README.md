# 🚀 Swingby Simulator

重力アシストシミュレーター - 4次ルンゲ・クッタ法による数値計算

## 概要

このウェブアプリケーションは、C言語で実装された重力アシストシミュレーション（src.c）をベースに、ブラウザ上で動作するインタラクティブなシミュレーターです。

## 特徴

- **物理シミュレーション**: 4次ルンゲ・クッタ法による高精度な軌道計算
- **リアルタイム可視化**: HTML5 Canvasによる軌道の描画
- **パラメータ調整**: 初期位置・速度の調整が可能
- **データエクスポート**: CSVファイルのダウンロード機能

## 物理モデル

地球の重力場における人工衛星の運動を計算します：

```
F = -GMm/r² (万有引力の法則)
```

定数：
- G: 6.67430×10⁻¹¹ m³/kg/s² (万有引力定数)
- M: 5.972×10²⁴ kg (地球の質量)

## ローカル実行

```bash
npm install -g serve
npm run dev
```

ブラウザで http://localhost:3000 を開く

## Vercelへのデプロイ

```bash
# Vercel CLIをインストール
npm install -g vercel

# デプロイ
vercel --prod
```

## 使い方

1. 初期条件（位置・速度）を設定
2. 「シミュレーション開始」ボタンをクリック
3. 軌道の変化をリアルタイムで観察
4. 必要に応じてCSVデータをダウンロード
